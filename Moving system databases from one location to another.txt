Moving System database from one location to another:

1) Start the instance in single/master only mode.

net stop "SQL Server (MSSQLSERVER)

net start "SQL Server (MSSQLSERVER) /m /t3608

2) Execute the commands for all system databases.

Resource logical names are "Data" and "Log" and database name is "mssqlsystemresource".

alter database mssqlsystemresource modify file(name='Data',filename='F:\Program Files\Microsoft SQL Server\MSSQL.1\MSSQL\mssqlsystemresource.mdf')
alter database mssqlsystemresource modify file(name='Log',filename='F:\Program Files\Microsoft SQL Server\MSSQL.1\MSSQL\mssqlsystemresource.ldf')


3) Move the files physically and update the Startup parameters Master MDF/LDF locations (ONLY IF MASTER DATABASE IS MOVED).

4) Stop and Start instance normally.
